This assignment is a shared canvas drawing app where multiple clients can connect and draw an image together on a shared canvas.
To be able to draw a user must enter a username and colour they wish to draw with and then simply just click their mouse on the canvas and draw.
---------------------------------------------------------------------------------------------------------------------------------------------------------

HOW TO COMPILE and run***
---------------------------------------------------------------------------------------------------------------------------------------------------------
Step 1: Open your command prompt and open the Assignment2 Directory
Step 2: install all needed npm modules
Step 3: in your command prompt type in node app.js
Step 4: go to link http://localhost:3000/sharedCanvas.html on 2 different browsers side by side
Step 5: enter your username and colour on both browsers
Step 6: Draw on the canvas by clicking on your mouse and simply drawing.
